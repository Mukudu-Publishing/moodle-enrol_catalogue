# Enrol Plugin: Catalogue #

__About__

An example of a enrolment plugin developed for the book and course.

__Description__

The plugin allows users to self enrol on any course that has this enrolment method enabled.
The catalogue of courses is accessed by following a links somewhere on the site to enrol/catalogue/index.php.
